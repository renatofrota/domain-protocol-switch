# domain-protocol-switch
A script to replace hardcoded occurrences of a domain, protocol (http/https) and prefix (www/no-www) in static files (assets, cache, etc)

## How to use

From the website root directory, run:

```
curl -sO https://raw.githubusercontent.com/renatofrota/domain-protocol-switch/master/domain-protocol-switch.bash && bash domain-protocol-switch.bash
```
